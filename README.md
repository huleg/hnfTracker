# hnfTracker
A hand and finger tracking repository that ultilizes OpenCV to track the user's hands and fingers.
Dependencies:
	> python3.4
	> python3.4-numpy
	> OpenCV 3.0.0
	> v4l2 (to disable auto exposure on linux webcam)
